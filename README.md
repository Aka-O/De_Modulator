# De_Modulator
This is a telecommunications project where I implemented a Frequency Modulation/Demodulation circuit, taking in a MATLAB signal as input and displayed on an oscilloscope. Later, the circuit was modified to also perform Phase Modulation/Demodulation. 

This repository will contain the MATLAB files that generate a signal to be fed into the circuit. It will also contain several LTSpice simulations, a report summarizing design choices and challenges faced, as well as pictures/videos documenting progress. The photos and videos may be stored in the google drive instead of here (see main README). 

Created for the course ELEC4123 | Design Proficiency at UNSW.
